Game!


Description:

Game! is a website that provides information about video games. Users can search video games and browse video games by categories. The website recommends games based on ratings and also provides similar games of the game users search.

Deployment:

Run the python file flask_app.py to start the application.

Features:

● The website displays a number of recommended games based on the user and critic ratings.
● The website displays the top three publishers, platforms and best-sellers based on the user and critic ratings.
● Users can browse video games by genre, platform, publisher, platform and sale record.
● Users can search for video games by name or publisher.
● The website provides a list of products by the same publisher as the searched video game.
● Users can connect with the developers through email.

Features that are not implemented:

● Users can create personal accounts and manage their lists of favourite games.
● The website can provide personalized recommendations of video games based on the users’ lists of favourite games.

Scripts for Database Table:

DROP TABLE IF EXISTS video_game;
CREATE TABLE video_game (
     Name text,
     Platform text,
     Year_of_Release text,
     Genre text,
     Publisher text,
     NA_Sales real,
     EU_Sales real,
     JP_Sales real,
     Other_Sales real,
     Global_Sales real,
     Critic_Score real,
     Critic_Count real,
     User_Score text,
     User_Count real,
     Developer text,
     Rating text
);
